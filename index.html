Arduino

ESP8266

The ESP8266 WiFi Module is a self contained SOC with integrated TCP/IP protocol stack that can give any microcontroller access to your WiFi network. The ESP8266 is capable of either hosting an application or offloading all Wi-Fi networking functions from another application processor. Each ESP8266 module comes pre-programmed with an AT command set firmware, meaning, you can simply hook this up to your Arduino device and get about as much WiFi-ability as a WiFi Shield offers (and that’s just out of the box)! The ESP8266 module is an extremely cost effective board with a huge, and ever growing, community.

Communications to ESP8266

AT commands over serial communication are used to communicate with the ESP8266. You can refer the complete command set from the instruction set reference

Basic AT commands we use in this tutorial are,

AT+RST
Used to reset the ESP8266, clearing RAM.

ATE0
Used to disable echo of last entered command.

AT+CWMODE=<mode>
Used to enable/disable client and AP mode of ESP8266.
<mode> values are,
1 : station mode
2 : softAP mode
3 : softAP + station mode

AT+CIPSERVER=<mode>,<port>
Used to initialise a TCP server on ESP8266.
<mode> and <port> values are,
<mode> 0 Delete server
<mode> 1 Create server
<port> port number, default is 333

AT+CIPSTART=<link ID>,<type>,<remote IP>,<remote port>
Used to Establish TCP connection or UDP transmission.
<link ID> ID of network connection (0~4), used for multi-connection
<type> string, "TCP" or "UDP"
<remote IP> string, remote IP address
<remote port> string, remote port number

AT+CIPSEND=<link ID>,<length>
Used to send data <link ID> ID of the connection (0~4), for multi-connect
<length> data length, MAX 2048 bytes

AT+CWJAP=<ssid>,<pwd>
Used to connect to an AP
<ssid> string, AP’s SSID
<pwd> string, MAX: 64 bytes ASCII

AT+ CIFSR
Used to get local IP address

AT+CWSAP=<ssid>,<pwd>,<chl>,<ecn>
Used for the configuration of softAP mode
<ssid> string, ESP8266 softAP SSID
<pwd> string, range: 8 ~ 64 bytes ASCII
<chl> channel id
<ecn>
0 OPEN
2 WPA_PSK
3 WPA2_PSK
4 WPA_WPA2_PSK
